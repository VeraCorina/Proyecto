este proyecto es para manejar redes de interacciones entre proteinas usando grafos y algoritmos como dijkstra para ver rutas metaboicas.
tambien permite cargar y guardar todo el laberinto de datos en archivos csv para que el profe no nos baje puntos por perder la info.
